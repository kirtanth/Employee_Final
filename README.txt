-------------------------------Welcome To Angular Json CRUD app---------------------------------------------------------------------

This is simple application where you can add delete edit and read entered data

Enjoy the Application




---------------------------How to start application------------------------------------------------------------------
Open IDE(Recommended: VS Code)
Go to Terminal -> Go in project folder
Fire Command : npm install (it takes 5 min)
		
Fire Command : json-server --watch src/assets/db.json (keep this terminal as it is this is json server)

Add new Terminal : ng serve
